# Continuum Mechanics — Cheatsheet
> 参考: Landau Vol.7, Timoshenko

---

## 应力-应变

### Hooke 定律 (各向同性)
```
σ = λ tr(ε)I + 2με
ε = σ/(2μ) - λ tr(σ) I / (2μ(3λ+2μ))
```

### 弹性常数互转
| 从 | E,ν | λ,μ | K,G |
|----|-----|-----|-----|
| E | E | μ(3λ+2μ)/(λ+μ) | 9KG/(3K+G) |
| ν | ν | λ/(2(λ+μ)) | (3K-2G)/(2(3K+G)) |
| G | E/(2(1+ν)) | μ | G |
| K | E/(3(1-2ν)) | λ+2μ/3 | K |

### 平面问题
平面应力: Q = E/(1-ν²)[1 ν 0; ν 1 0; 0 0 (1-ν)/2]
平面应变: D = λ[1-ν ν 0; ν 1-ν 0; 0 0 (1-2ν)/2]/((1+ν)(1-2ν))

## 弹性波

```
P波: v_p = √((λ+2μ)/ρ)
S波: v_s = √(μ/ρ)
Rayleigh: v_R ≈ (0.862+1.14ν)/(1+ν) v_s
Love: v_s1 < v < v_s2
```

反射系数 (垂直入射): R = (Z₂-Z₁)/(Z₂+Z₁), Z=ρc

## 梁理论

### Euler-Bernoulli
```
EI d⁴w/dx⁴ = q(x)
简支中点: w_max = 5qL⁴/(384EI)
悬臂端点: w_max = PL³/(3EI)
临界屈曲: P_cr = π²EI/(KL)²
```

### K因子
两端铰支 K=1.0, 一端固定 K=2.0, 两端固定 K=0.5, 固定-铰支 K=0.7

## 板

```
弯曲刚度: D = Eh³/(12(1-ν²))
圆板 (简支): w_max = qa⁴(5+ν)/(64D(1+ν))
圆板 (固支): w_max = qa⁴/(64D)
简正频率: ω_mn = ((mπ/a)²+(nπ/b)²)√(D/ρh)
```

## 强度准则

```
von Mises:  σ_vm = √[½((σ₁-σ₂)²+(σ₂-σ₃)²+(σ₃-σ₁)²)] = √(3J₂)
Tresca:     max(|σ₁-σ₂|,|σ₂-σ₃|,|σ₃-σ₁|)
Rankine:    max(σ₁,σ₂,σ₃)
Mohr-Coulomb: τ = c + σ_n tan φ
```

## 疲劳

```
Basquin: σ_a = σ_f'(2Nf)^b
Coffin-Manson: ε_a = (σ_f'/E)(2Nf)^b + ε_f'(2Nf)^c
Goodman: σ_a/σ_ar + σ_m/σ_uts = 1
Gerber:  σ_a/σ_ar + (σ_m/σ_uts)² = 1
Miner:   D = Σ n_i/N_fi ≤ 1
```
